This file contains metadata for both the Stationstatus table.
For more information, see the contest page.

Variables:
- id: unique id
- station_id: station id
- update: timestamp (yyyy-mm-dd hh-mm-ss)
- nbBikes: number of avaible bikes
- nbEmptyDocks: number of empty docks
- capacity: capacity of station (the number of bikes that a station can hold, sum of nbBikes and nbEmptyDocks)

Notes:
* Data is aggregated from data found at: http://thehubway.com/data/stations/bikeStations.xml
* Data collection started at 8/22/2011
* Data for 9/21/2011 to 11/9/2011 is missing
